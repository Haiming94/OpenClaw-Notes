# Skills

Skills 之与 OpenClaw 的关系，就像方法论之与工具的关系：OpenClaw 是让 AI “能干活”的引擎，而 Skills 是让它 “干好活” 的方法论。二者结合，才能实现从“聊天 AI”到“执行型 AI Agent”的真正跨越。

因此，Skills 是 OpenClaw 的核心内容，也是 OpenClaw 的灵魂所在。本文分享如何安装 Skills，提升 OpenClaw 的能力。同时，也分享一些我正在使用的 Skills，以供大家参考。

## 安装 Skills

这里介绍三种方式：clawhub(npm)，手动下载 和 自动安装。

### 使用 clawhub 安装 skills

这是最推荐的方式，通过 ClawHub 官方仓库安装：

```bash
# 1. 搜索可用的 skills
clawhub search <关键词>

# 2. 查看 skill 详情
clawhub view <skill-name>

# 3. 安装 skill
clawhub install <skill-name>

# 4. 更新已安装的 skills
clawhub update

# 5. 列出已安装的 skills
clawhub list

# 6. 卸载 skill
clawhub uninstall <skill-name>
```

**注意事项：**

- 安装前确保 OpenClaw 已正确配置
- 某些 skills 可能需要额外的配置（如 API 密钥）
- 定期检查更新以保持安全和功能最新

### 手动下载安装 skills

适用于本地开发的 skills 或无法通过 clawhub(npm) 安装的场景：

```bash
# 1. 下载或克隆 skill 仓库
git clone <skill-repo-url>
# 或下载 zip 文件后解压

# 2. 进入 skill 目录
cd <skill-directory>

# 3. 验证 skill 结构
# 确保包含 SKILL.md 和必要的 scripts/、references/ 等目录

# 4. 安装到 OpenClaw
clawhub install ./<skill-directory>

# 5. 验证安装
clawhub list
```

**Skill 目录结构示例**：

> my-skill/
> ├── SKILL.md          # 必需：技能描述和使用说明
> ├── scripts/          # 可选：辅助脚本
> ├── references/       # 可选：参考文档
> └── examples/         # 可选：使用示例

**从 zip 文件安装：**

```bash
# 解压 zip 文件
unzip my-skill.zip -d my-skill

# 安装
clawhub install ./my-skill
```
